# UniPilot SL — Varsity Day Manager

A high-fidelity, single-viewport daily dashboard built specifically for university students in Sri Lanka to track schedules, gaps, deadlines, and transit options.

## ✨ Features
- **Dynamic Departure Board:** Real-time feedback on your current or upcoming class countdowns.
- **Break Time Tracker:** Auto-calculates gaps between lectures for optimal campus study blocks.
- **Zero-Dependency Core:** Built using vanilla Semantic HTML5, modern CSS3 Flex-Matrix layouts, and clean ES6 JavaScript.
- **Privacy-First Sandboxing:** Saves data persistently using browser local storage without tracking overhead.

## 🛠️ Tech Stack
- Frontend: Vanilla JS, HTML5, CSS3 Variables
- Platform Hooks: Vercel Static Hosting Engine
